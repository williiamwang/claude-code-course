# Claude Code 最佳实践课程

一个交互式的 Claude Code 学习课程网站，包含：

- 📚 **12 个阶段的学习课程** - 从基础到高级
- 💬 **AI 导师答疑功能** - 回答课程相关问题
- 🌐 **在线部署** - 部署到 Vercel 免费托管

## 快速开始

### 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000

### 部署到 Vercel

1. 推送代码到 GitHub
2. 在 Vercel 导入项目
3. 添加环境变量 `MINIMAX_API_KEY`
